# Song Recommendation System

## Overview
Machine Learning based Spotify song recommendation system that suggests similar songs using multiple ML algorithms and audio features.

## Features
- Content-based recommendation using Cosine Similarity
- K-Means clustering for song grouping
- K-Nearest Neighbors (KNN) recommendation
- Genre-based song recommendations
- Artist-based recommendations
- Interactive recommendation menu

## Machine Learning Techniques Used
- K-Means Clustering
- K-Nearest Neighbors (KNN)
- Cosine Similarity
- Feature Scaling using StandardScaler

## Libraries Used
- Pandas
- NumPy
- Scikit-learn

## Dataset
Spotify songs dataset

## Project Highlights
- Audio feature preprocessing
- ML model training
- Recommendation generation
- Similarity analysis

## Note
This is a backend-based mini project focused on machine learning recommendation logic.
